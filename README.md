# Go custom implementations
This project includes custom implementations of different **sync tools** provided by Golang. Each folder contains the implementation and the use in a sample application.

**Note:** The goal of this project to learn inner workings of Go sync tools. The use of of these tools are discourged in real world projects.

Each tool has its respective Medium article explaining its inner workings.

1- [Implement our own RWMutex](https://medium.com/@hasnatinter10/golang-implement-our-own-rwmutex-0eac04243cbd)

2- [Implement a custom write-preferred Mutex](https://medium.com/@hasnatinter10/golang-implement-a-custom-write-preferred-mutex-c4901ee3d945)

3- [Implement a custom Wait Group](https://medium.com/@hasnatinter10/golang-implement-a-custom-waitgroup-631fa349a8ee)
